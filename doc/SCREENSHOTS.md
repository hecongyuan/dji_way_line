# 项目截图说明

## 📸 截图列表

### 1. 主界面 (main_interface.png)
**对应聊天中的：图片1**
- 展示主地图界面
- 左侧：航线库面板，显示已创建的航线列表
- 右侧：3D地图视图（基于高德地图+Cesium）

- 右上角：地图控制按钮
- 推荐尺寸：1920x1080 或 1600x900

**保存位置**：`doc/images/main_interface.png`

### 2. 创建航线对话框 (create_mission.png)
**对应聊天中的：图片2**
- 展示"创建新航线"对话框
- 选择航线类型：
  - 点状航线（选中状态，蓝色高亮）
  - 线状航线
  - 多边形航线
  - 仿地航线
  - 矩形网格航线
- 限定区域航线设置
- 选择飞行器：Matrice 4 系列、3D 系列、4E等
- 选择挂载：Matrice 4E、Matrice 4T（蓝色选中）
- 航线名称输入框
- 底部：取消和确定按钮
- 推荐尺寸：800x600

**保存位置**：`doc/images/create_mission.png`

### 3. 航线编辑界面 (route_editor.png)
**对应聊天中的：图片3**
- 展示航线编辑和参数配置界面
- 左侧面板：
  - 顶部：导入KMZ / 生成KMZ 按钮
  - 航线统计信息（总距离：921m，飞行时长：3分24秒，覆盖面积：22301 m²）
  - 三个标签页：任务配置、锁点航线、原点列表
  - 任务配置选项：
    - 飞防航点点模式/完成动作
    - 安全模式/自动返航
    - 失控动作/航线失航行动
    - 航线模式/全局动作
    - 航线高度/无
    - 安全都飞高度(m)：20
    - 全局飞行速度(m/s)：5
    - 航点高度模式：海拔高度/相对飞点/相对地形
- 右侧地图：
  - 显示已绘制的航线（黄色路径）
  - 多个红色航点标记
  - S形扫描路径
  - 地图显示校园/建筑区域
- 推荐尺寸：1920x1080 或 1600x900

**保存位置**：`doc/images/route_editor.png`

## 📋 操作步骤

### 方式一：手动保存（推荐）
1. 在聊天界面保存上传的3张截图
2. 创建目录：`mkdir -p doc/images`
3. 将截图按照上述对应关系重命名并保存到 `doc/images/` 目录

### 方式二：重新截图
如果需要更高质量的截图，建议在应用运行时重新截图：

```bash
# 启动应用
pnpm dev

# 打开浏览器访问 http://localhost:5173
# 按照上述说明截取相应界面
```

**截图建议**：
- 使用全屏截图工具（macOS: Cmd+Shift+3/4，Windows: Win+Shift+S）
- 确保截图清晰，文字可读
- 统一使用PNG格式（除微信二维码用JPG）
- 建议使用1920x1080或更高分辨率

## 📝 注意事项

1. **隐私保护**：截图中如果包含敏感信息（坐标、位置），请确认是否可以公开
2. **文件大小**：PNG截图建议压缩后小于500KB，保持清晰度的同时减小仓库体积
3. **命名规范**：使用小写字母和下划线，与README中的引用保持一致
4. **Git提交**：记得将 `doc/images/` 目录添加到Git并提交

```bash
git add doc/images/
git commit -m "docs: add application screenshots"
```
